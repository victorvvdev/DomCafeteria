# Dom Cafeteria ☕

Projeto desenvolvido como Requisito Funcional para a disciplina de Ciência da Computação na **UNIFOR**, com o objetivo de criar uma plataforma web para divulgação e interação com a cafeteria fortalezense **Dom Cafeteria**.

---

##  Sobre o Projeto

O sistema busca apresentar de forma acessível e atrativa a identidade da cafeteria, permitindo que usuários conheçam os produtos, a história do estabelecimento e realizem interações diretas, como avaliações e agendamentos.

### Principais Funcionalidades
* **Vitrine Digital:** Exposição de imagens dos principais cafés, pratos e do ambiente.
* **Nossa História:** Seção dedicada à trajetória e ao processo de criação da marca.
* **Cardápio Interativo:** Visualização de itens com mecanismo de avaliação de preferências.
* **Agendamento de Encomendas:** Canal direto para solicitação de pedidos e informações de contato.

---

##  Público-Alvo

O sistema foi projetado para:
* Moradores de Fortaleza e visitantes interessados em gastronomia local.
* Pessoas que buscam ambientes para lazer, encontros sociais ou trabalho remoto.

---

##  Tecnologias Utilizadas

A stack tecnológica do projeto foi selecionada para garantir performance, responsividade e organização de dados:

* **HTML5:** Estruturação de conteúdo.
* **CSS3 & Bootstrap:** Estilização da interface, garantindo padronização visual e responsividade.
* **JavaScript:** Implementação de interatividade e dinamismo nas páginas.
* **Node.js:** Execução de código no lado do servidor e gerenciamento de comunicação.
* **MySQL:** Sistema de gerenciamento de banco de dados relacional para armazenamento das informações.
